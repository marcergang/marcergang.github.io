Summary: Latitude vs X

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this this analysis, we first pulled data from OpenWeatherMap API to assemble a dataset over 500 cities. After assembling the dataset, we used Matplotlib to plot various aspects of the weather vs latitude. Factors included: tempature, cloudiness, wind speed and humidity. This site provides the source data and visualization created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
